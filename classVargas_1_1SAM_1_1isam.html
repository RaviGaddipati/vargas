Vargas::SAM::isam Class Reference

#include <sam.h>

Detailed Description

Provides an interface to read a SAM file.

Minimal error checking is done, no error will be raised if there is a mismatch (e.g. a ref name that is not defined in the header).
See Vargas::SAM for more usage information.
