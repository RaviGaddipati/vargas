Vargas::ByteAligner::AlignmentGroup Class Reference

#include <alignment.h>

Detailed Description

Container for a packaged batch of reads.

Reads are interleaved so each SIMD vector contains bases from all reads, respective to the base number. For example AlignmentGroup[0] would contain the first bases of every read. All reads must be the same length. Minimal error checking.

Template Parameters
SIMDPP_FAST_INT8_SIZEmax number of reads. If a non-default T is used, this should be set to SIMDPP_FAST_T_SIZE where T corresponds to the width of T. For ex. Default T=simdpp::uint8 uses SIMDPP_FAST_INT8_SIZE
