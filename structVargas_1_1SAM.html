Detailed Description

Provides structures representing SAM data.

SAM files can be either read with isam, or written with osam. Both use the same core structures to store data. Each SAM file consists of:

  • 1 Header with:
    • Any number of ReadGroups
    • Any number of Sequences
    • Any number of Programs
  • Any number of Records Tags defined in the spec can be directly accessed. Other tags are stored in an aux Optional struct.
    No conflict management is done.
