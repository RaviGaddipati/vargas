Vargas::Aligner< num_reads, CellType, NativeT >::AlignmentGroup Class Reference

#include <alignment.h>

Detailed Description

template<unsigned int num_reads = SIMDPP_FAST_INT8_SIZE, template< unsigned int, typename=void > class CellType = simdpp::uint8, typename NativeT = uint8_t>
class Vargas::Aligner< num_reads, CellType, NativeT >::AlignmentGroup

Container for a packaged batch of reads.

Reads are interleaved so each SIMD vector contains bases from all reads, respective to the base number. For example AlignmentGroup[0] would contain the first bases of every read. All reads must be the same length. Minimal error checking.

Template Parameters
num_readsmax number of reads. If a non-default T is used, this should be set to SIMDPP_FAST_T_SIZE where T corresponds to the width of T. For ex. Default T=simdpp::uint8 uses SIMDPP_FAST_INT8_SIZE
