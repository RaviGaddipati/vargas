Detailed Description

template<unsigned int core_size>
class dyn_bitset< core_size >

Dynamic bitset backed by fixed bitsets.

Groups together multiple fixed bitsets to emulate a dynamic bitset. A vector of std::bitset<core_size> maintains the information.
