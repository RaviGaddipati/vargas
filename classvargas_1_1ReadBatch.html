Detailed Description

template<unsigned int num_reads = SIMDPP_FAST_INT8_SIZE, template< unsigned int, typename=void > class T = simdpp::uint8>
class vargas::ReadBatch< num_reads, T >

Container for a packaged batch of reads.

Reads are interleaved so each SIMD vector contains bases from all reads, respective to the base number. For example ReadBatch[0] would contain the first bases of every read. Short reads or missing reads are padded with Base::N.

Template Parameters
num_readsmax number of reads. If a non-default T is used, this should be set to SIMDPP_FAST_T_SIZE where T corresponds to the width of T. For ex. Default T=simdpp::uint8 uses SIMDPP_FAST_INT8_SIZE
