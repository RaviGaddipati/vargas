const std::vector< std::string > & vargas::VarFile::genotypes ( )

Get a list of alleles for all samples (subject to sample set restriction).

Consecutive alleles represent phasing, e.g. all odd indexes are one phase, all even indexes are the other. Call will unpack the full record. Explicit copy number variations are replaced, other ambigous types are replaced ambiguous. A map is also built, mapping each allele to the subpopulation that has it.

Returns
Vector of alleles, ordered by sample.
